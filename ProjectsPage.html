VazeNeon

About

A First Person Shooter with parkour movement mechanics. It's heavily inspired by games like Halo and Karlson. It's my first project that I intend to finish and release, and hopefully make better a sequel. I haven't been able to come up with a storyline for the campaign yet as currently my main focus is on fine tuning the mechanics, adding more features, making more enemy variants etc. I have a trello board for this project where I keep track of my progress, you can check it out using the link given below.

I have released this game on itch.io, where you can download and test the game for yourself. I keep updating the itch page as I work on the game.

Check out the Trello board

- Windows 7,8,10,11
